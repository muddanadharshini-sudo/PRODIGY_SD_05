# Web Scraping
## Description
A Python program that extracts product information from an e-commerce website and stores it in a CSV file.
## Features
- Extracts product names
- Extracts product prices
- Extracts product ratings
- Saves the data to a CSV file
## Technologies Used
- Python
- Requests
- BeautifulSoup
- CSV
## How to Run
1. Install the required libraries:
   ```bash
   pip install requests beautifulsoup4
   ```
2. Run:
   ```bash
   python web_scraper.py
   ```
## Output
The scraped data is stored in `books.csv`.
